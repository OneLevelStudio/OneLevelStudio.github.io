Project AMADEUS

Storing a human's "brain" (behavior + knowledge + memory) in an interactive digital format.
The name "AMADEUS" inspired from a science-fiction series "Steins;Gate", which is an advanced program that uses the data of a person's memories and personality as base, capable of creating a fully functional AI with all the capabilities and memories of the person whose data is using.
Our initial idea: Human Thinking → Brainwaves → Digital Format → "Brain" Dataset → LLM Fine-tuning → Interactive AI
